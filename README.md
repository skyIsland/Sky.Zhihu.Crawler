### 抓取知乎问题的回答里的图片
1. #### 解压Bin压缩包
2. #### 打开Setting.json进行编辑参数
3. #### 双击运行 Sky.ZhiHuImage.Crawler.exe
#### 参数说明
###### Cookie:顾名思义,就是你知乎账号的Cookie了
###### QuestionId:问题ID
###### SaveSrc:图片保存路径
###### SaveCount:一个文件夹保存多少图片
### 文件夹说明
1. Sky.ZhiHu.Crawler文件夹 为源码
2. Data文件夹为sqlite数据库,仅保存图片的路径和生成的唯一码
